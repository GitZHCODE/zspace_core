Detailed Description

Collection of methods for form finding using thrust network analysis ( 2D Graphic Statics).

Based on Block, Philippe, and John Ochsendorf. "Thrust network analysis: A new methodology for three-dimensional equilibrium." Journal of the International Association for shell and spatial structures 48.3 (2007): 167-173.
