This method offsets the boundary faces of the input mesh based on vertex color. It uses only the red channel of the color.

face offset based on http://pyright.blogspot.com/2014/11/polygon-offset-using-vector-math-in.html

Parameters
[in]keepExistingFaces- true if existing face needs to be retained.
[in]minVal- minimum offset. Needs to be between 0 and 1.
[in]maxVal- maximum offset. Needs to be between 0 and 1.
[in]useVertexColor- true if vertex color is to be used else face color is used.
