vector< double > &  influences,
vector< T > &  fieldValues,
double  power = 2.0,
bool  normalise = true 
)

This method computes the field values based on inverse weighted distance from the input positions.

Template Parameters
T- Type to work with double / zVector.
Parameters
[in]fieldMesh- input field mesh.
[in]inPositions- container of input positions for distance calculations.
[in]values- value to be propagated for each input position. Size of container should be equal to inPositions.
[in]influences- influence value of each input position. Size of container should be equal to inPositions.
[out]fieldValues- container for storing field values.
[in]power- input power value used for weight calculation. Default value is 2.
[in]normalise- true if the scalars need to mapped between -1 and 1. generally used for contouring.
Since
version 0.0.1
