This method computes the horizontal equilibrium of the form and force diagram.

Based on Block, Philippe, and John Ochsendorf. "Thrust network analysis: A new methodology for three-dimensional equilibrium." Journal of the International Association for shell and spatial structures 48.3 (2007): 167-173.

Parameters
[in]computeTargets- true if the targets fro horizontal equiibrium have to be computed.
[in]formWeight- weight of form mesh update. To be between 0 and 1.
[in]dT- integration timestep.
[in]type- integration type - zEuler or zRK4.
[in]angleTolerance- angle tolerance for parallelity.
[in]minMax_formEdge- minimum value of the target edge for form mesh.
[in]minMax_forceEdge- minimum value of the target edge for force mesh.
[in]printInfo- prints information of minimum and maximum deviation if true.
[in]colorEdges- colors diagram edges based on deviation if true.
Returns
bool - true if the all the correponding edges are parallel.

This method computes the vertical equilibrium of the result diagram using iterative method.

Parameters

[in]type- integration type - zEuler or zRK4.
[in]forceDiagramScale- scale of force diagram.
[in]tolerance- tolerance for force check.
Returns
bool - true if the all the forces per vertex add up to zero or below tolerance.
